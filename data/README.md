# Recorded-data access metadata

Recorded RGB frames and WT61C-TTL CSV files are distributed outside GitHub. Request access through the [EgoWrist Gesture 30 Dataset Access Request](https://forms.office.com/r/EqiJNLWgtF). `download_links.json` is the machine-readable registry for the request URL and any published external archive versions, sizes, and checksums.

The data package contains both wrist-camera views and IMU measurements. The expected hierarchy and validation workflow are documented in [`../docs/DATA_ACCESS.md`](../docs/DATA_ACCESS.md) and [`../docs/USAGE.md`](../docs/USAGE.md).
